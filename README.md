# Photo Albums

A simple app to look albums, their photos and details of every photo.

Using jetpack compose, MVVM and faking api response with https://jsonplaceholder.typicode.com/ 

## Libraries used
- Hilt
- Retrofit
- Coil 
- Accompanist Navigation Animation
